Nevers Now

» React Native - Symfony 4

Création d'une application mobile permettant de trouver des lieux dans la ville de Nevers, comme des monuments ou des places de stationnement pour personnes à mobilité réduite. L'application est développée en React Native. Elle est rattachée à une API développée en PHP sur Symfony 4 ansi qu'à l'API Google Maps. Réalisé en collaboration avec Sylvain Espina Magdalena, Thomas Galluser et Romain Leroy.
